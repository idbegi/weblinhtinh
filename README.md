# NewsApp

Ứng dụng Đọc Tin Tức tích hợp back-end (Spring Boot) và front-end (HTML/CSS/JavaScript).

## Hướng dẫn chạy

1. **Yêu cầu:**
   - Java 11+
   - Maven

2. **Chạy ứng dụng:**
   ```
   mvn spring-boot:run
   ```

3. **Truy cập giao diện:**
   - Mở trình duyệt tại: http://localhost:8080/index.html
